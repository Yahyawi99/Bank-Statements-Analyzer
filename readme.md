# Overview

The FinTech industry is really hot right now. Mark Erbergzuck
realizes that he spends a lot of money on different purchases and
would benefit from automatically summarizing his expenses. He
receives monthly statements from his bank, but he finds them a bit
overwhelming. He has tasked you with developing a piece of
software that will automate the processing of his bank statements so
he can get better insights into his finances. Challenge accepted!

# Single Responsibility Principal

1. Reading input
2. Parsing the input in a given format
3. Processing the result
4. Reporting a summary of the result
